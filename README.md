## **MyNotes app**
Приложение для управления заметками, разработанное с использованием современных Android-технологий и архитектурных подходов.

## **Технологический стек:**

- **Язык**: Kotlin  
-  **UI**: Jetpack Compose  
-  **Архитектура**: MVVM, Clean Architecture  
-  **Асинхронность**: Coroutines, Flow, StateFlow  
-  **Локальная база данных**: Room  
-  **DI**: Hilt  

## Архитектура и ключевые решения

**Clean Architecture с разделением на слои**
- **Domain** - бизнес-логика и use cases
- **Data** - работа с данными (Room)
- **Presentation** - UI (Compose) и ViewModel

##  Функциональность

-  Создание, редактирование, удаление заметок
-  Поиск по заметкам
-  Локальное хранение данных (Room)
-  Реактивный интерфейс с мгновенными обновлениями
-  Clean Architecture для масштабируемости
